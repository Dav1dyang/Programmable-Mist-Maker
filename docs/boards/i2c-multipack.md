# I2C MultiPack Kit

!!! info "In design"
    This board is being designed right now —
    [watch the repo](https://github.com/Dav1dyang/Programmable-Mist-Maker) to know the
    moment it's real.

Drive a whole cloud: **chainable I2C mist controllers with six PWM channels per
node**, built for installations and multi-emitter builds. Each node hangs off the
Qwiic/I2C bus, so one XIAO can orchestrate many mist emitters.

## Planned

- 6 PWM mist channels per node
- Chainable over I2C / Qwiic
- Designed to pair with the [Battery Kit](battery-kit.md)'s Qwiic expansion port
